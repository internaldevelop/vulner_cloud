附录：基本知识
IEC-101: 厂站与调度主站间通讯，串行，多采用非平衡传输
IEC-102: 电量主站与站内抄表终端通讯
IEC-103: 与站内几点保护设备间通讯
IEC-104: 厂站与调度主站间通讯，以太网，多采用平衡传输

IEC60870—5—101是符合调度端要求的基本远动通信规约；
IEC60870—5—104是用网络方式传输的远动规约；
IEC60870—5—102是用作电能量传送的通信规约；
IEC60870—5—103是为继电保护和间隔层（IED）设备与站控层设备间的数据通信传输的规约。
